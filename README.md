# vue-todo

## Project setup
```
1. (npm이 설치 안 된 경우)

npm install

2. 프로젝트 Clone 후 node_modules 다운로드

npm i

3. Vuex 설치

npm i vuex@3.6.2 --save
```

### Compiles and hot-reloads for development

```
npm run serve
```

```
npm run build
```
